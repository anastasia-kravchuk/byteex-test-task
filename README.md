## Getting started

---

1. Clone the repository:
```bash
git clone https://github.com/anastasia-kravchuk/byteex-test-task.git
cd byteex-test-task
```
2. Install dependencies:
```bash
npm install
# or
yarn install
```
Run the project locally:
```bash
npm run dev
# or
yarn ren dev
```
